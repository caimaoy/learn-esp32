# LLM Wiki Schema - ESP32 学习

## 目录结构
```
wiki/
├── index.md           # 索引（所有页面列表 + 一句话描述）
├── log.md             # 日志（追加记录，格式: ## [日期] 操作 | 标题）
├── esp32-overview.md  # 开发板概述
├── first-blink.md     # Blink 项目详解
├── serial-communication.md  # 串口通信概念
├── setup-guide.md     # 开发环境搭建
└── ...                # 新增页面根据内容扩充
```

## 约定

### 索引
- 每次新增/修改页面后更新 `index.md`
- 按类别分组：概念、指南、常见问题

### 日志格式
```
## [YYYY-MM-DD] 操作 | 标题
- 具体记录内容
```
操作类型：记录、整理、问答、改造

### 命名规范
- 文件名：小写 + 连字符
- 标题：中文 # 一级标题
- 引用 wiki 内其他页面使用相对路径 `[标题](page.md)`

### 维护规则
- 索引、日志、内容页面三者可以在一次修改中完成
- 内容页不需要注释说明来源——来源由日志记录
