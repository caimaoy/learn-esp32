# PWM (脉冲宽度调制)

## 概念

PWM 通过快速开关引脚，用**占空比**模拟不同电压。

```
占空比 0%:  ▁▁▁▁▁▁▁▁  (0V)
占空比 50%: ▁▁▁▁▆▆▆▆  (≈1.65V)
占空比 100%: ▆▆▆▆▆▆▆▆  (3.3V)
```

## ESP32 的 LEDC (LED PWM Controller)

ESP32 有 16 个独立 PWM 通道 (0-15)，每个通道可独立设置频率和占空比，互不干扰。

```
通道 0 → GPIO 2 (LED, 5000Hz)
通道 1 → GPIO ? (蜂鸣器, 1000Hz)
通道 2 → GPIO ? (舵机, 50Hz)
```

## 关键参数

| 参数 | 说明 |
|------|------|
| **频率** | 每秒开关次数，LED 用 5000Hz 足够（人眼无闪烁） |
| **分辨率** | 占空比的精度，8 位 = 0-255，10 位 = 0-1023 |
| **占空比 (duty)** | 高电平占比，0 = 全灭，最大值 = 全亮 |

## 核心函数

```cpp
ledcSetup(channel, freq, resolution);  // 配置通道参数
ledcAttachPin(pin, channel);           // 绑定 GPIO 到通道
ledcWrite(channel, duty);              // 设置占空比
```

`ledcAttachPin` 的作用是把**物理引脚**接到**内部 PWM 发生器**上——换引脚只需重新 attach，通道不变。
